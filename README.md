# Africa Wildfire Analysis (Google Earth Engine)

This project analyzes wildfire activity across Africa using MODIS satellite data. It includes temporal and spatial analysis of active fires, burned areas, and Fire Radiative Power (FRP), with an interactive interface for visualization.

##  Key Features
- Processing large satellite datasets
- Detection of active fire pixels
- Burned area analysis
- Fire Radiative Power (FRP) analysis
- Interactive layer selection and visualization
- Temporal filtering of satellite imagery

##  Technologies
Google Earth Engine, JavaScript, MODIS satellite data

## Key Concepts
Remote sensing, big geospatial data, temporal analysis, spatial visualization

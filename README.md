
# Africa Wildfire Analysis (Google Earth Engine)

Wildfire activity analysis across Africa using MODIS satellite data in Google Earth Engine.  
Includes an interactive map interface to visualize active fires, burned areas, and Fire Radiative Power (FRP).

##  Story Map
[Interactive project presentation](https://storymaps.arcgis.com/stories/xxxxx)

##  Datasets
- MODIS Active Fire (MOD14A1)  
- MODIS Active Fire (MYD14A1)  
- MODIS Burned Area (MCD64A1)

##  Analysis Includes
- Active fire pixels  
- Fire Radiative Power (FRP)  
- Burned areas  
- Interactive layer switching: MOD active fire, MYD active fire, Burned area, MOD FRP, MYD FRP

##  Tools
Google Earth Engine, JavaScript, MODIS satellite data
